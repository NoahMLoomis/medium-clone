## About
This is a blog clone (medium ish), made to practice NextJS, and a few other technologies

## Functionality
A user can:
- Create an account
- Sign In/Out
 - View/Write blog posts
 

## Getting Started
```bash
npm install

npm run dev
```
## Tech Stack
- __Frontend__: NextJS, React-TS
- __Backend__: Supabase
- __Styling__: TailwindCSS + DaisyUI
